## Project Overview

    SunCart is a modern summer-themed eCommerce web application where users can explore and purchase seasonal products such as sunglasses, summer outfits, skincare items, and beach accessories.

    The platform provides a smooth user experience with authentication, protected routes, and a responsive design for all devices.

##  Key Features

    1. User Authentication (Email/Password + Google Login)
    2. Browse Summer Products
    3. Product Details Page (Protected Route)
    4. User Profile Page
    5. Update Profile Information
    6. Fully Responsive Design (Mobile, Tablet, Desktop)
    7. Modern UI with Tailwind CSS & DaisyUI
    8. Animated UI using Lottie
    9. Fast Performance with Next.js App Router

##  Tech Stack

    1. Framework: Next.js
    2. Styling: Tailwind CSS + DaisyUI
    3. Authentication: BetterAuth
    4. Animation: Lottie React
    5. Deployment: Vercel

##  Authentication System

    Email & Password Login
    Google Social Login
    Protected Routes (Product Details)
    Redirect after login
    🌐 Environment Variables

### Create a .env.local file and add:

    NEXT_PUBLIC_AUTH_URL=your_auth_url
    NEXT_PUBLIC_GOOGLE_CLIENT_ID=your_google_client_id
    AUTH_SECRET=your_secret_key

### ⚠️ Never expose secret keys in public repositories.
